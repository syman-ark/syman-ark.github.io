    折叠面板

    Collapse-折叠面板

    将内容区域折叠/展开

    API

    属性名类型默认值描述最低版本
    titleLabelString面板的标题
    isOpenBooleanfalse折叠

    slot

    名称说明
    name定制自己需要的内容

    效果图

    示例

    //collapse.axml
<view class="page">
  <collapse titleLabel="{{titleLabel}}" isOpen="{{isOpen}}">
  <view slot="content">
    //定制的内容
    <view class="custom">
      <text>标签名</text>
      <text>内容名</text>
    </view>
  </view>
  </collapse>
</view>

    //collapse.js
Page({
  data: {
    titleLabel:"面板",
    isOpen:true
  },
  onLoad() {},
});

    //collapse.json
{
   "defaultTitle": "collapse",
  "usingComponents": {
  "collapse": "@syman/ark-dingtalk/src/collapse/index"
   
  }
}
