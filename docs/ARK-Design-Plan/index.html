    ARK Design Plan

    ARK Design的立意

    方向：面向B端产品提供服务。

1:是由于伯俊软件提供服务的目标客户的行业性质所决定；

2:C端产品受众群体范围广且产品类别差异化较大，无法量化设计方案；

3:从UI（User Interface）到UE的（User Experience）的全范围覆盖；

    目的：

1:帮助团队节省大量的开发时间和精力；甚至中小企业都无需另请设计即可完成界面设计，大大缩小开发成本；

2:串联起ARK生态圈，实现产品界面从构思到落地的一站式服务；

3:为设计师站寻找更优的交互解决方案，完善用户需求，提升用户体验提供有力的支持，而不仅仅局限于界面样
式；


    宗旨：

1:完善ARK生态体系；

2:设计工作可流程化，提高效率；

3:帮助企业决策；

    ARK Design规划大纲

    1-3月：基础资源填充；

4-5月：实现在线预览（除ARK Mobile外所有资源在线可视化）；

6月：  落实对外展示出口 + 用户可使用ARK Design的方案； 3:从UI（User Interface）到UE的（User 
Experience）的全范围覆盖；
