时间轴

TimeLine-时间轴

垂直的展示一系列信息

API

属性名类型默认值描述最低版本
titleString时间轴的标题
isShowLineBooleantrue是否显示线
nodeLineTopString-32rpx线的top的定位

效果图

示例

//timeline.axml
<view class="page">
  <view class="heading">时间轴支持卡槽</view>
    <timeline a:for="{{editRecordList}}" title="{{item.title}}">
    <view slot="content" class="timeLineItem">
    //卡槽的名字固定写成content
     定制为自己卡槽内容
    </view>
  </timeline>
</view>

//timeline.js
Page({
  data: {
    editRecordList:[{
      title:'内容一'
    },{
      title:"内容二"
    }]
  },
  onLoad() {},
});

//timeline.json
{
"defaultTitle": "timeline",
  "usingComponents": {
    "timeline": "@syman/ark-dingtalk/src/timeline/index" //引入组件
   
  }
}
