R3系统配置URL跳转规则

菜单跳转定制界面

根据以下配置参数所生成的路由： http(s)://domain:port/CUSTOMIZED/FUNCTIONPERMISSION/2305

配置参数
作为定制界面路由的ID使用id2305
作为Tab名称使用label内部定制menu
定制界面配置规则urlCUSTOMIZED/FUNCTIONPERMISSION(定制界面英文名称)
定制界面的路由组成说明
固定标识SYSTEM
单对象固定标识TABLE_DETAIL
结构形式（上下结构V/左右结构/H）V
主表表名DL_B_PUR
主表ID23792
界面为新增的固定标识New

菜单跳转指定单对象新增界面

根据以下配置参数所生成的路由： http(s)://domain:port/SYSTEM/TABLE_DETAIL/V/DL_B_PUR/23792/New

配置参数
配置的路由urlSYSTEM/TABLE_DETAIL/V/DL_B_PUR/23792/New
单对象新增界面的路由组成说明
固定标识SYSTEM
单对象固定标识TABLE_DETAIL
结构形式（上下结构V/左右结构/H）V
主表表名DL_B_PUR
主表ID23792
界面为新增的固定标识New

菜单跳转外链界面

根据以下配置参数所生成的路由： http(s)://domain:port/LINK/MENUOUT/2302

配置参数
作为外链界面路由的ID使用id2302
作为Tab名称使用label外链menu
网页链接urlhttps://www.baidu.com/
外链界面名称作为路由拼接使用valuemenuout (可配置任意英文名称)
外链类型的路由组成说明
外链界面固定标识LINK
外链名称MENUOUT
外链界面ID2302
