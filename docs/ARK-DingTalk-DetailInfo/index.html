详情模板布局

DetailInfo-详情模板

主要从来展示类似于表单信息的

API

属性名类型默认值描述最低版本
listArray渲染的数据
totalColumnNumber2展示数据的列数

效果图

示例

//detailinfo.axml
<view class="page">
  <detail-info list="{{companyInfoList}}">
  
  </detail-info>
  <view class="header">卡槽形式的用法</view>
  <detail-info list="{{testSlot}}">
  <view slot="test">
    我是定制部分的内容
  </view>
  </detail-info>
</view>
