    JFlowPlugin Version

    Introduce

    JlofwPlugin是打通中台业务系统以及Jflow流程控制系统的插件，能够在Jlofw后台配置系统中配置流程图,在中台业务系统中进行流程控制。JflowPlugin是基于R3业务系统开发，目前JflowPlugin基本R3业务系统区分了1.3版本和1.4版本

    1.3版本

    $ npm install jflow-plugin --save

    import jflowplugin,{components} from 'jflow_plugin'
Vue.use(jflowplugin, {
    router,
    axios,
    store,
    jflowIp: '',  //jflow后台地址
    showTab: false   //是否启用自定义待办列表(默认采用弹窗形式展示)
  });

    components待办列表实例，支持只定义待办列表

    1.4版本

    $ npm install burgeon-r3 --save

    BurgeonR3的配置项中配置jflow:

    ParamTypeDescriptionDefault Value
    enableJflowBoolean是否启用JFlow插件false
    jflowRequestDomainStringJFlow后台地址
    showTabBoolean是否启用自定义待办列表false

    详情请看R3.launchApplication
